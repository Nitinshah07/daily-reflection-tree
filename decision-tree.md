# Daily Reflection Decision Tree

## Input
User enters current mood:
- happy
- sad
- confused
- stressed

---

## Flow

Happy
→ Ask what went well today
→ Suggest gratitude reflection

Sad
→ Ask what caused the feeling
→ Suggest journaling or talking to someone

Confused
→ Ask what choices are available
→ Suggest pros and cons analysis

Stressed
→ Ask source of stress
→ Suggest breaking tasks into smaller steps

---

## Guardrails
- Only valid moods accepted
- Invalid input asks user to retry
- Fixed deterministic outputs
